# book-reader

## **Purpose:**
Fullstack book reader utilizing Reactjs, Flask, and GraphQL. Created separate backends testing Flask and Graphql server. 

## **Frontend:** 
- **Home Page**
  - Used GraphQL to fetch pages. 
  - Search bar for book search
  - Clickable list of books
  - Working navbar
